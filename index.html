1. Naam aanpassen

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de master branch zit.

  • Vervang bovenaan de pagina "Naam van student" door jouw eigen naam (id=naam)
  • Commit deze wijziging

2. Afbeeldingen toevoegen

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de master branch zit.

  • Maak een nieuwe branch: afbeeldingen
  • Controleer dat je in deze branch zit
  • Zoek een passende afbeelding voor dit hoofdstuk (id=afbeeldingen), voeg deze afbeelding toe in de map images.
  • Pas de src aan van de placeholder afbeelding van het introductie hoofdstuk aan, zodat de nieuw toegevoegde afbeelding zichtbaar.
  • Commit deze wijziging
  • Ga terug naar de master branch.
  • Merge de aanpassingen van afbeeldingen naar master.
  • Push de master branch naar github

3. Een conflict oplossen

Winnaar: Superman

Hulk

  • Zorg dat je in de master branch zit
  • Begin een nieuwe branch: hulk
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Hulk
  • Commit deze wijziging.

Superman

  • Ga terug naar de master branch
  • Begin een nieuwe branch: superman
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Superman
  • Commit deze wijziging.

Conflict!

  • Ga terug naar de master branch
  • Merge de aanpassingen van superman naar master.
  • Controleer of superman de winnaar is (refresh de pagina).
  • Merge de aanpassingen van hulk naar master.
  • Conflict!

    Je hebt nu onderstaand conflict in de header van dit article.

    
<<<<<<< HEAD
<span id="winnaar">Superman</span>
=======
<span id="winnaar">Hulk</span>
>>>>>>> hulk
  • Zorg dat Hulk de winnaar is.
  • Commit de merge
  • Push de master branch naar github

4. Een conflict oplossen

Soms is er bij een conflict geen keuze, maar moet je handmatig de wijzigingen samen voegen.

Donaties: € 0,00

Gulle gever

  • Zorg dat je in de master branch zit
  • Begin een nieuwe branch: gulle-gever
  • Pas het hierboven aangegeven donatiebedrag (id=donaties) aan naar: 100,00
  • Commit deze wijziging.

Dagobert Duck

  • Ga terug naar de master branch
  • Begin een nieuwe branch: dagobert-duck
  • Pas het hierboven aangegeven donatiebedrag (id=donaties) aan naar: 1,99
  • Commit deze wijziging.

Conflict!

  • Ga terug naar de master branch
  • Merge de aanpassingen van dagobert-duck naar master.
  • Controleer of de donatie 1,99 is (refresh de pagina).
  • Merge de aanpassingen van gulle-gever naar master.
  • Conflict!

    Je hebt nu onderstaand conflict in de header van dit article.

    
<<<<<<< HEAD
<span id="donaties">€ 1,99</span>
=======
<span id="donaties">€ 100,00</span>
>>>>>>> gulle-gever
  • Zorg dat de donaties bij elkaar worden opgeteld.
  • Controleer of het donatiebedrag correct is (refresh de pagina).
  • Commit de merge
  • Push de master branch naar github

5. Versie update

Versiebeheer eindigt natuurlijk altijd met een nieuwe versie.

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de master branch zit.

  • Verhoog de versie (id=version) met 0.1
  • Commit en push de wijziging
  • Lever de URL van jouw github repository in.
